Woolwork

Turn any website into wool. Woolwork is a drop-in design system where every surface is felt, every border is a stitch, and every control is a real sewn object: buttons settle when pressed, checkboxes get cross-stitched shut, cards arrive by settling onto the board and having their stitch drawn on. This page is running it right now.

Try it

Check the box slowly and watch the thread sew across in one stroke per press phase. Every card on this page arrived by settling onto the board and having its stitch drawn around it, once, on load.

Install

One stylesheet, one script, two fonts. No build step, no framework requirement. The visuals are plain CSS and the behavior attaches once at the document level, so it works the same in static HTML, server-rendered pages, and single-page apps, on elements that exist now or get added later.

<link href="https://fonts.googleapis.com/css2?family=Baloo+2:wght@600;700;800&family=Nunito:wght@400;600;700;800&display=swap" rel="stylesheet">
<link rel="stylesheet" href="/woolwork/woolwork.css">
<script src="/woolwork/woolwork.js" defer></script>
<body class="board"> ... </body>

Building new

Every common UI element has a craft counterpart ready to use: inputs are pockets, accordions are zippers, modals are pinned notes, dividers are seams. Start from the component page and compose.

Restyling existing sites

Woolwork changes how your site looks, feels, and moves without touching its structure, copy, or flows. Your layout stays; the material changes.

Made for AI agents
